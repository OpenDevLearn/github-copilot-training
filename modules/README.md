# Modules

Week-by-week advanced training modules.

Each module folder contains:
- `README.md` - Deep-dive content, mental models, patterns
- `LAB.md` - Hands-on exercises with acceptance criteria
- `CHECKLIST.md` - Review checklist, pitfalls, anti-patterns
- `examples/` - Runnable code/config snippets

## Modules

| Week | Module | Focus |
|------|--------|-------|
| 1 | [Copilot + CLI](week1-copilot-cli/) | Automation, natural-language Git, batch refactors |
| 2 | [Custom Agents](week2-custom-agents/) | Persona definition, autonomy bounds, multi-step planning |
| 3 | [Skills](week3-skills/) | Reusable behaviors, dependency graphs, composition |
| 4 | [MCP Server](week4-mcp-server/) | Custom tools/APIs, sandboxing, streaming responses |
