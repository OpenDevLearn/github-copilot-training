# Labs

Hands-on, repo-first exercises with binary acceptance criteria.

Each lab includes:
- **Objective**: Measurable outcome
- **Prerequisites**: Required knowledge/setup
- **Tasks**: Step-by-step with acceptance criteria
- **Verification**: How to validate completion
- **Evidence**: Artifacts to commit (logs, configs, transcripts)

## Evidence Requirements

All labs must produce repo-committed artifacts:
- Configuration files
- Agent logs (redacted)
- Test results
- Architecture diagrams (Mermaid)
- Prompt transcripts (sanitized)
